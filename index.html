Mes projets

Livr'Échanges

  • Projet étudiant mené seul, site web avec comme thème principal le don de livres
  • Technologies utilisées : PHP, JavaScript, HTML, CSS, MySQL
  • Projet codé selon l'architecture MVC (Modèle-Vue-Contrôleur)
  • Le site comprend une base de données de livres, les utilisateurs peuvent laisser des commentaires et une note pour chaque livre, en plus de s’inscrire comme donateur.
  • Le site comprend un système de messagerie pour que les utilisateurs se mettent d’accord sur la modalité de la donation ou discuter.
